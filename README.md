# edge-zapret-gui

Графический интерфейс для удобного запуска и обновления [zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube?tab=readme-ov-file)

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
